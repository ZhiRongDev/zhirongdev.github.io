專案

  • RAG & LangGraph 聊天機器人

    網頁開發

    前往專案:

    此為個人 Side Project，實作一套結合 RAG、LangGraph 與 Web Search API 的 LLM 聊天平台。 前往 GitHub:

    • 前端使用 Vue 3 + TypeScript 建立聊天介面
      • 提供即時聊天介面，支援一般 LLM 與 RAG 回覆模式。
      • 支援 Markdown 與程式碼高亮顯示，提升 AI 回應可讀性。
      • 使用 Pinia 與 Vue Router 管理狀態與頁面流程。
      • 透過 VeeValidate + Yup 進行表單驗證，避免不合法輸入。
      • 使用 Vitest 與 Playwright 進行單元與 E2E 測試。
    • 後端使用 FastAPI 建立高效能 LLM 與 RAG API 服務
      • 整合 LangGraph Agent，依照使用者問題動態判斷是否需要進行 Web Search。
      • 串接 Gemini File Search API，實作文件上傳、向量化、語意檢索與 RAG 回覆。
      • 支援多家 LLM Provider（Gemini / OpenAI / Anthropic）。
      • 使用 JWT 與 bcrypt 實作使用者驗證與權限控管。
      • 提供 Email 驗證與忘記密碼流程，透過 Gmail SMTP 發送驗證信與重設連結。
      • 使用 SQLModel + PostgreSQL 建立關聯式資料模型，並透過 Alembic 管理資料庫遷移。
      • 使用 Redis 實作滑動視窗 Rate Limiter，限制使用者請求頻率。
      • 基於 Pydantic 進行型別驗證並自動產生 Swagger API 文件。
      • 使用 pytest 撰寫後端測試，確保核心功能穩定性。
    • 系統架構與部署
      • 使用 Docker 與 Docker Compose 建立開發與正式環境。
      • 正式環境透過 Nginx 處理反向代理與靜態資源。
      • 整合 Zeabur CI/CD，GitHub Push 後自動建置與部署。

    前往專案:

    本專案為一套 No-Code 數據分析與機器學習平台，透過直覺化操作取代繁瑣程式撰寫，快速完成數據探索、視覺化、模型訓練與預測。系統支援多種圖表與 AI 分析能力，並提供自動化機器學習訓練、分析與預測，涵蓋分類、回歸、聚類等多種任務，協助使用者高效做出精準決策。

      • 支援 i18n 多國語系，網頁可以進行中英文切換。

      • 使用 Plotly 進行資料視覺化。
      • 使用 Vitest 撰寫測試。
      • 串接 Payuni 金流。
    • 後端使用 Flask 以及 FastAPI 提供 API，包含機器學習、 LLM 圖表分析、RAG 聊天機器人等服務，負責部分程式碼撰寫以及除錯的工作。
    • 應用 Docker 於程式開發以及部署。
    • 服務部署於 AWS EC2。

  • Acer Sense Ring 智慧戒指雲端服務

    網頁開發

    本專案無法提供網址。

    Acer FreeSense Ring 智慧戒指 的雲端資料同步服務，用於上傳與查詢用戶的健康資料紀錄，並提供管理者後台監看資料上傳狀況。

    • 前端使用 Vue.js 建立後台介面。
      • 設計對使用者友善的操作介面。
      • 防範 XSS 攻擊，包含登入憑證使用 Cookie 儲存避免被竊取。
      • 使用 Gmail SMTP 實作忘記密碼的寄信服務。
    • 後端使用 FastAPI、SQLModel、PostgresSQL 等工具建立穩定的 API 服務:
      • 串接 Acer 的 Single Sign-On 註冊服務，並透過 Redis 管理登入 Session。
      • 基於 Pydantic 進行型別驗證、自動產生 Swagger API 文件。
      • 使用 Gunicorn 設定合理的 Uvicorn worker 提高 API 吞吐量、提高服務穩定性(根據 CPU Core 與資料庫連線池)。
      • 透過 Python 的平行任務模組處理 CPU 瓶頸，並善用快取與非同步的機制處理 I/O 瓶頸、避免服務阻塞。
      • 使用 Locust 進行 API 壓測，在只有 4 vCPUs 的情況下可以達到 303.61 RPS 與不到 0.2% 的失敗率，且服務全程可用 (模擬最高同時 2000 人持續使用服務 10 分鐘)。
      • 使用 FastAPI Typer 建立用於專案的 CLI 工具。
      • 使用 Pytest 測試後端 API。
    • 配置 Nginx 處理靜態資源、SSL 加解密、服務轉發、流量限制等。
    • 服務部署於 AWS EC2。
